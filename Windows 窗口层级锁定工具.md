# Windows 窗口层级锁定工具教程

本工具通过 AutoHotkey 脚本修改窗口的 `WS_EX_NOACTIVATE` 样式，实现“点击窗口但不触发置顶”的效果。

---

## 1. 下载与安装环境

在使用脚本之前，你需要安装 **AutoHotkey (AHK)** 解释器：

* **官方下载地址**：[https://www.autohotkey.com/](https://www.autohotkey.com/)
* **推荐版本**：选择 **v2.0**（当前最新的稳定版本）。
* **安装说明**：下载后运行安装程序，按提示完成安装即可，无需额外配置。

---

## 2. 脚本代码

创建一个后缀为 `.ahk` 的文件（例如 `LockWindow.ahk`），并将以下内容粘贴进去：

```autohotkey
; ==============================================================================
; 功能：锁定/解锁窗口层级（点击不置顶）
; 快捷键：Ctrl + Alt + S
; ==============================================================================

#Requires AutoHotkey v2.0

^!s::
{
    ; 获取当前鼠标位置下的窗口句柄
    try {
        MouseGetPos ,, &id
        
        ; 获取窗口当前的扩展样式 (ExStyle)
        ; 0x08000000 代表 WS_EX_NOACTIVATE
        exStyle := WinGetExStyle("ahk_id " id)
        
        if (exStyle & 0x08000000) {
            ; 移除样式
            WinSetExStyle("-0x08000000", "ahk_id " id)
            ToolTip "✅ 窗口已恢复正常 (点击会置顶)"
        } else {
            ; 添加样式
            WinSetExStyle("+0x08000000", "ahk_id " id)
            ToolTip "🔒 窗口层级已锁定 (点击不置顶)"
        }
    } catch Error {
        ToolTip "❌ 操作失败：权限不足或窗口无效"
    }
    
    ; 3秒后自动消失提示框
    SetTimer () => ToolTip(), -3000
}
```

---

## 3. 使用步骤

1.  **运行脚本**：双击 `LockWindow.ahk` 文件。任务栏右下角出现绿色 **H** 图标表示运行成功。
2.  **准备环境**：打开一个窗口（如记事本），用另一个窗口（如浏览器）盖住它的一部分。
3.  **激活锁定**：将鼠标悬停在**被遮挡的那个窗口**上，按下 `Ctrl + Alt + S`。
4.  **操作测试**：此时你用鼠标点击该窗口，它会响应点击（如点击菜单），但**绝不会**跳到最前面遮挡当前窗口。

---

## 4. 常见问题 (FAQ)

* **无法输入文字？**
    该模式下窗口不获取“焦点”，因此键盘无法打字。如需输入，请再次按下快捷键解锁。
* **对某些窗口无效？**
    如果目标窗口（如任务管理器、游戏）是以管理员权限运行的，请**右键点击脚本 -> 以管理员身份运行**。
* **想开机自启？**
    按下 `Win + R` 输入 `shell:startup`，将脚本的快捷方式放入该文件夹。
